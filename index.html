Proyectos

SMILE

Es una PWA responsiva que está enfocada en promover el apoyo psicológico en nuestra sociedad. En este proyecto utilizamos Vue con Vite como frameworks de frontend utilizando el lenguaje de Typescript, y en backend únicamente node.js con MySQL para el registro de usuarios. Se hizo uso de una API de podcasts para darle funcionalidad a un reproductor en uno de los módulos de podcasts relacionados con Psicología.

Se utilizó otra API para frases de motivación aleatorias, la biblioteca de sockets.io para chats grupales en tiempo real, y falta por completar la correcta implementación de localización de centros psiquiátricos cercanos al usuario.

Repositorio de GitHub: car-mor
DestinoMX

Gracias a este proyecto pude aprender demasiado sobre el framework de javascript Vue, ya que fui parte del equipo de front-end, aunque de igual manera estuve dándole funcionalidad a las pantallas de Descripción del lugar, Favoritos, e Historial, al hacer uso de la API de Google Places de Google Maps. Fue una gran oportunidad de trabajar con un equipo grande de 12 integrantes.

En este proyecto reforcé mis habilidades con Git y Github para subir nuestros cambios y descargar los de los demás miembros, así como el seguimiento de una metodología de desarrollo que fue SCRUM, y Kanban, ya que parte de nuestro flujo de trabajo fue el uso de Trello para la visualización de actividades.

World Happiness Report

Este proyecto es un caso práctico que realicé gracias al curso que tomé de Análisis de Datos de Google, y en este lo primero que hice fue crear un nuevo conjunto de datos usando dos fuentes distintas que contenían información sobre los distintos factores que influyen en la felicidad alrededor del mundo desde el año 2015 al 2023. Al tener este nuevo conjunto de datos, realicé la limpieza, y selección de los datos que ocupé para una visualización que hice después en Tableau Public.
